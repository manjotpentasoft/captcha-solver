## Recognition Tasks

Recognition tasks use AI models to analyze and extract information from different types of CAPTCHA content, including images, text prompts, and audio challenges.

These tasks are ideal when you need to **interpret CAPTCHA data and return a human-readable result**, such as extracted text or transcribed audio.

---

## Image CAPTCHA Recognition

Use this task type when the CAPTCHA consists of **distorted text or numbers embedded in an image**.

### When to Use
- OCR-style image CAPTCHAs
- Visual verification challenges
- Image-based CAPTCHA prompts

### Task Behavior
- The result is returned **synchronously**
- You receive the recognized text directly in the API response

This task is best suited for fast, real-time image CAPTCHA solving.

---

## Text CAPTCHA Parsing

Use this task type when the CAPTCHA presents a **textual challenge** that requires understanding and generating a response.

### When to Use
- Text-based verification questions
- Logic or prompt-style CAPTCHA challenges

### Task Behavior
- Processed synchronously
- Returns an AI-generated text response

This task is useful for CAPTCHAs that rely on **human-like text comprehension**.

---

## Audio CAPTCHA Transcription

Use this task type when the CAPTCHA is delivered as **audio**, typically for accessibility purposes.

### When to Use
- Audio CAPTCHAs
- Accessibility-focused verification flows

### Task Behavior
- May require short processing time
- Result is returned as a transcribed text response

This task is commonly used to support **accessibility workflows** and automated testing.
